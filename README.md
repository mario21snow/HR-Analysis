# HR Analytics Dashboard using Power BI

This dashboard is designed to provide valuable insights into HR datasets, helping decision-makers plan their organizational strategy effectively. Below, I'll give you an overview of the topics covered in this Power BI project and the visualizations included in the dashboard.

## Topics Covered in Power BI Project:

### Use Power BI

This Dashboard will walk you through the Power BI interface and its various components, ensuring you have a strong foundation in navigating and utilizing this powerful tool. Learn about its intuitive features and functions within Power BI Desktop.

### Importing Data in Power BI

Dive into data importing in Power BI, covering how to import data from diverse sources like Excel workbooks, CSV files, and databases. This knowledge will enable you to effectively bring data into Power BI for analysis.

### Power Query in Power BI

Gain a deep understanding of Power Query in Power BI and how to transform the data. Explore functions such as data cleaning, adding columns, changing data types, and replacing values, ensuring your data is prepared for analysis.

### Creating Charts in Power BI

Utilize the wide range of charts and graphs available in Power BI to visualize the data effectively. Learn how to craft the data into visual forms that convey insights clearly.

### Filters and Slicers in Power BI

Filters and slicers are essential for interactively exploring your data and gaining deeper insights. Learn how to use these features effectively to refine the data analysis.

### Designing Dashboards in Power BI

Learn how to design interactive dashboards that provide a comprehensive view of the data, making it easier for the audience to understand and interact with the insights.
